# 市场主题研究报告

本 skill 将一个研究主题转化为五维结构化报告。整个过程分四个阶段执行，阶段二中五个维度的数据采集**并行**进行以节省时间。

---

## 阶段一：确定研究宇宙（必须先执行）

在拉任何数据之前，先明确研究对象的边界。

### 1.1 识别板块/主题类型

判断用户主题属于哪种分类体系，然后找到对应的成分股：

| 主题类型 | 定位方法 | 成分股接口 |
|---------|---------|-----------|
| 申万行业（如"半导体"、"医药"） | `index_classify` 查行业代码 | `index_member_all` |
| 同花顺概念（如"ChatGPT概念"） | `ths_index` 搜索 | `ths_member` |
| 中信行业 | `ci_daily` 查代码 | `ci_index_member` |
| 自定义股票池 | 用户直接给出 ts_code 列表 | — |

### 1.2 确定研究参数

```
研究主题：<用户输入>
成分股列表：<ts_code 列表，最多取前30只流通市值最大的>
板块指数代码：<对应的申万/同花顺/中信指数代码>
时间范围：
  - 行情/资金/消息：近3个月（start_date = 今天-90天）
  - 基本面：近12个月（或最近4个季度）
  - 宏观：近24个月
```

---

## 阶段二：并行数据采集

**五个维度同时开始采集**，不要顺序等待。每个维度的数据写入单独的 DataFrame 或文件，供阶段三分析使用。

---

### 维度 A：行情面

**目标**：板块整体走势 + 个股表现分化

```python
# 板块指数日线（3个月）
sw_daily(index_code=板块代码, start_date=..., end_date=...)      # 申万行业指数
# 或
index_daily(ts_code=指数代码, start_date=..., end_date=...)

# 个股日线（前复权）
pro_bar(ts_code=ts_code, adj='qfq', start_date=..., end_date=...)

# 每日基本面指标（PE/PB/换手率/市值）
daily_basic(ts_code=ts_code, start_date=..., end_date=...)

# 技术面因子（可选，若需要MACD/RSI等）
stk_factor_pro(ts_code=ts_code, start_date=..., end_date=...)
```

**关键输出**：
- 板块近3个月涨跌幅 vs 同期沪深300
- 成分股涨幅排名 TOP10 / BOTTOM10
- 近期换手率、成交量变化趋势
- 当前估值水位（PE/PB历史百分位）

---

### 维度 B：资金面

**目标**：了解谁在买卖、力度如何

```python
# 板块资金流向（同花顺）
moneyflow_ind_ths(ts_code=板块代码, start_date=..., end_date=...)

# 概念板块资金流向
moneyflow_cnt_ths(ts_code=概念代码, start_date=..., end_date=...)

# 个股主力资金流（抽样重点个股）
moneyflow_ths(ts_code=ts_code, start_date=..., end_date=...)
# 或
moneyflow_dc(ts_code=ts_code, start_date=..., end_date=...)

# 北向资金（沪深港通总体）
moneyflow_hsgt(start_date=..., end_date=...)

# 北向资金十大成交股（看是否有板块个股入榜）
hsgt_top10(trade_date=最近交易日)

# 融资融券（板块整体情绪）
margin_detail(ts_code=成分股ts_code, start_date=..., end_date=...)
```

**关键输出**：
- 板块近3个月资金净流入/流出累计金额及趋势
- 北向资金对该板块的持续买入/卖出信号
- 融资余额变化（杠杆资金的态度）
- 是否有龙虎榜异动个股

---

### 维度 C：基本面

**目标**：行业整体估值与盈利质量

```python
# 财务指标（近4个季度，抽取重点个股）
fina_indicator(ts_code=ts_code, start_date=近一年)
# 关注：roe, eps, grossprofit_margin, debt_to_assets

# 利润表（营收增速）
income(ts_code=ts_code, period=最近季报期)

# 业绩预告/快报（判断景气度趋势）
forecast(ann_date=近90天)
express(ann_date=近90天)

# 券商盈利预测（市场一致预期）
report_rc(ts_code=ts_code, start_date=近90天)
# 关注：eps_p2、np_p2（预测净利润）是否上调

# 机构调研（机构关注热度）
stk_surv(ts_code=ts_code, start_date=近90天)
```

**关键输出**：
- 板块平均 PE/PB 及历史分位
- 主要成分股 ROE、营收增速
- 近期业绩预告方向（预喜/预亏比例）
- 券商盈利预测是否在上调

---

### 维度 D：消息面

**目标**：近期重要事件与市场情绪

```python
# 新闻快讯（主题关键词过滤）
news(src='sina,eastmoney', start_date=近30天, end_date=今天)
# 过滤含主题关键词的条目

# 长篇通讯（深度新闻）
major_news(start_date=近30天)

# 券商研究报告（近期发布）
research_report(ts_code=重点个股ts_code, start_date=近60天)

# 上市公司公告（重大事项）
anns_d(ts_code=成分股, start_date=近30天)
# 重点看：定增、回购、股东减持、重大合同

# 互动问答（投资者关心的问题）
irm_qa_sz(ts_code=ts_code, start_date=近30天)  # 深市
irm_qa_sh(ts_code=ts_code, start_date=近30天)  # 沪市
```

**关键输出**：
- 近30天重大新闻事件时间线（3-5条最重要的）
- 最新券商研报评级与目标价
- 近期重大公告摘要（定增/回购/减持/合同）
- 政策利好/利空信号

---

### 维度 E：宏观面

**目标**：与主题相关的宏观环境

根据研究主题选择相关的宏观指标（不必全部拉取）：

| 主题类型 | 最相关宏观数据 |
|---------|-------------|
| 消费/零售 | CPI (`cn_cpi`)、社零增速 |
| 工业/制造 | PPI (`cn_ppi`)、PMI (`cn_pmi`) |
| 金融/地产 | M2 (`cn_m`)、社融 (`sf_month`)、LPR (`shibor_lpr`) |
| 出口/外贸 | 汇率 (`fx_daily`)、美债收益率 (`us_tycr`) |
| 通用背景 | GDP (`cn_gdp`)、PMI (`cn_pmi`)、M2 (`cn_m`) |

```python
# 示例：消费主题
cn_cpi(start_m='2年前', end_m='当月')    # 月度数据，格式 YYYYMM
cn_pmi(start_m='2年前', end_m='当月')

# 货币环境（所有主题通用）
shibor(start_date=近6个月)
cn_m(start_m='2年前', end_m='当月')
```

**关键输出**：
- 与主题最相关的2-3个宏观指标的趋势判断
- 当前宏观环境对该板块是顺风/逆风/中性

---

## 阶段三：分析原则

在综合五个维度时，遵循以下判断逻辑：

**共振分析**（最有价值）：
- 行情上涨 + 资金持续净流入 + 业绩上调 = 强共振信号
- 行情下跌 + 资金持续净流出 + 业绩下调 = 弱势确认
- 行情下跌 但 资金逆势流入 + 估值处于低位 = 潜在机会

**分化分析**：
- 识别板块内部的强弱分化（哪些个股在领跑，为什么）
- 机构调研集中在哪几只个股（资金预埋伏信号）

**背离警示**：
- 行情上涨但资金流出 = 注意拉高出货
- 北向资金与A股主力反向 = 值得关注

## 注意事项

**API 调用限制**：
- Tushare 单次请求有行数上限（通常6000行），日线数据需按股票循环分批获取
- 成分股超过30只时，优先取流通市值前30
- 分钟级数据积分消耗较高，默认不拉取，除非用户明确要求

**数据缺失处理**：
- 如果某只股票某个接口返回空，跳过该股票，不影响整体分析
- 宏观数据有发布延迟（GDP季度更新、CPI月度更新），取最新可用期

**输出长度**：
- 执行摘要要精炼，整体报告控制在2000-3000字
- 数据用表格呈现，避免大段罗列原始数字
