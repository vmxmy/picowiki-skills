# 自选股组合删除


### 接口介绍
接口：p_del
描述：删除自选股组合


### 输入参数
名称 | 类型  | 必选 | 描述
---- | ----- | ---- | ----
name | int | Y | 组合名称



### 输出参数
名称 | 类型 | 默认显示 | 描述
---- | ---- | ---- | ----
status | bool | Y | 是否执行成功



### 代码示例
```python
# 删除组合(我的股票池)
df=pro.p_del(name="我的股票池")
```

### 数据结果
```text
       status
    0   True
```
