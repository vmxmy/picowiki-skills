# Tushare 接口列名差异与使用注意事项

> 来源：本地实测汇总（2026-04-10）
> 这些差异是高频 bug 来源，使用前必查。

---

## 一、涨跌幅列名不统一（最易混淆）

| 接口 | 涨跌幅列名 |
|------|----------|
| `daily` | `pct_chg` |
| `weekly` / `monthly` | `pct_chg` |
| `bak_daily` | `pct_change` |
| `sw_daily`（申万行业指数） | `pct_change` |
| `top_list`（龙虎榜） | `pct_change` |
| `index_daily` | `pct_chg` |

**规则**：个股日线/周线/月线用 `pct_chg`；申万行业指数、备用行情、龙虎榜用 `pct_change`。

---

## 二、市值/股本单位不统一

| 接口 | 字段 | 单位 |
|------|------|------|
| `daily_basic` | `circ_mv` / `total_mv` | **万元** |
| `bak_daily` | `float_mv` / `total_mv` | 元（需确认） |
| `bak_basic` | `float_share` / `total_share` | **亿** |
| `stock_basic` | — | 无市值字段 |
| `stk_premarket` | `total_share` / `float_share` | **万股** |
| `daily_basic` | `total_share` / `float_share` | **万股** |

**换算**：`daily_basic.circ_mv` 换亿元 = `/10000`

## 三、north_money 是余额不是净流入

`moneyflow_hsgt` 的 `north_money` 字段是**持仓余额（累计）**，不是当日净流入。

正确计算日净变化：
```python
df = pro.moneyflow_hsgt(start_date=..., end_date=...)
df = df.sort_values('trade_date')
df['net_inflow'] = df['north_money'].astype(float).diff()
```

## 四、index_member_all 返回全市场行业映射，不是单指数成分

```python
# 错误：以为能直接得到某指数成分
members = pro.index_member_all(index_code='801081.SI')

# 正确：需按 l2_code 或 l3_code 过滤
semi = members[members['l2_code'] == '801081.SI'][['ts_code', 'name']]
```

列结构：`l1_code, l1_name, l2_code, l2_name, l3_code, l3_name, ts_code, name, in_date, out_date, is_new`

## 五、sw_daily 返回全量，需后置过滤

```python
# sw_daily 不能真正按单代码过滤，会返回全市场申万指数
df = pro.sw_daily(index_code='801081.SI', start_date=..., end_date=...)
# 必须再过滤
df = df[df['ts_code'] == '801081.SI']
```

## 六、daily_basic 查询策略

按 `ts_code` 字符串批量查（超过约 20 只时）容易返回空。

**推荐方式**：按 `trade_date` 查全市场，再用成分股列表过滤：
```python
basics = pro.daily_basic(trade_date='20260410',
                         fields='ts_code,pe_ttm,pb,circ_mv,turnover_rate_f')
basics = basics[basics['ts_code'].isin(semi_list)]
```

## 八、bak_daily 的高价值特有字段

| 字段 | 说明 | 其他接口是否有 |
|------|------|--------------|
| `interval_3` | 近3月涨幅(%) | ❌ 仅 bak_daily |
| `interval_6` | 近6月涨幅(%) | ❌ 仅 bak_daily |
| `strength` | 强弱度(%) | ❌ |
| `activity` | 活跃度(%) | ❌ |
| `attack` | 攻击波(%) | ❌ |

用 `bak_daily` 一次查全市场 + `interval_3` 排序，可替代循环计算个股涨跌幅：
```python
bak = pro.bak_daily(trade_date='20260410',
                    fields='ts_code,name,interval_3,interval_6,pe,float_mv')
top = bak[bak['ts_code'].isin(semi_list)].sort_values('interval_3', ascending=False)
```
